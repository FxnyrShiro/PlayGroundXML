# MusicApp

A simple music search application built with Jetpack Compose, using the Deezer API.

## Features

* Search for tracks by name.
* View track details.
* Save favorite tracks to a local database.
* Share tracks with friends.

## Tech Stack

*   **UI:** Jetpack Compose
*   **Architecture:** MVI (Model-View-Intent) with Clean Architecture principles.
*   **DI:** Koin
*   **Networking:** Retrofit & OkHttp
*   **Database:** Room
*   **Navigation:** Jetpack Navigation for Compose
*   **Image Loading:** Coil
